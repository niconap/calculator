In this project I created a calculator using CSS, HTML and JavaScript.

Use the buttons with numbers to enter numbers, use an operator to do a certain calculation and press the "equals" button to get your answer!
Use the "C" button to clear the display and start over.

Skills shown in this project:
- Using HTML and CSS to make a webpage.
- Using JavaScript array methods.
- Using JavaScript functions.
- Using JavaScript to manipulate the DOM.
